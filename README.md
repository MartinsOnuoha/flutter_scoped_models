# Flutter Scoped Models

Simple Scoped Model Example in Flutter

## Installation

To Install Application

- `git clone https://github.com/MartinsOnuoha/flutter_scoped_models.git`
- `cd flutter_scoped_models`
- `flutter packages get`
- `flutter run`
